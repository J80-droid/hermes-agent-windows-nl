# Creative — procedures (SOP)

## Visual

1. Check merk/stijl in RAG (`search_knowledge`) vóór nieuw ontwerp
2. Statische assets: file + optioneel image_gen na J.-OK
3. Diagrammen: excalidraw-skill; export naar `output/research/` indien tijdelijk

## Motion

1. **Manim:** wiskundige/geometrische uitleg → `manim-video` skill
2. **Hyperframes:** motion-graphics, social, captions → install optional skill; DESIGN.md gate; `lint` → `preview` → `render`
3. Zware renders: GPU/duur met J. afstemmen; escalatie `ict` bij infra

## Interactive

1. TouchDesigner: `touchdesigner-mcp` skill; documenteer netwerk in `Interactive/`
2. p5js: `code_execution` optioneel na J.-vraag

## Writing

1. Brief in `Writing/` of chat; geen bindende publicatie zonder J.
2. `creative-ideation` voor brainstorm; humanizer alleen na expliciete vraag
