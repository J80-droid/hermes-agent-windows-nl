# Legal taxonomie — bronmappen

Nieuwe rechtsgebied-lens toevoegen:

1. Werk `docs/LEGAL_TAXONOMY.md` in de Hermes-repo bij.
2. Maak hier een submap: `../<SubmapNaam>/` (naast deze `_Taxonomy` map).
3. Werk `profiles\legal\SOUL.md` bij (tabel Juridische lenzen).
4. Draai `update_knowledge.bat legal`.

Zaak-mappen (bijv. `Geschillencommissie Rijk/`) zijn **geen** rechtsgebied — details in `LEGAL_ACTIVE_MATTERS.md`.

Zie repo: `docs/LEGAL_DOMAIN_ARCHITECTURE.md`.
